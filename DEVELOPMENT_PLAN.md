# 🚀 《时空星球》开发计划

> **版本**：v1.0
> **创建日期**：2026-06-14
> **基于文档**：PRD.md

---

## 📋 开发原则

1. **先前端后后端** — 前端先行，快速验证 UI 效果
2. **先基础后功能** — 先完成项目骨架和数据库搭建，再开发业务功能
3. **每步验收** — 每项任务完成后需向产品负责人汇报，验收后方可继续
4. **按模块开发** — 按功能模块逐个开发，确保每个模块完整可用
